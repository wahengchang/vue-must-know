it is a collection of examples, listing the handful functions and usecases for VUE development.

## Install
```
$ git clone git@github.com:wahengchang/vue-must-know.git
$ npm i
```

## Run
```
$ npm start
```

open browser `localhost:3000/1-basic.html`
